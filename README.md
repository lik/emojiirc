# emojiirc
IRC bot that responds to emoji shortnames (e.g. `:bear:`)

## Dependencies
 - [irc-client](https://github.com/johnelse/ocaml-irc-client)
and others probably

## Functionality
Currently, this bot parses all messages for emoji shortnames and returns the appropriate emojis.
